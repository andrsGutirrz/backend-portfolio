# Movie API

### Description

This is an open API that serves a list of movies that can be filtered using query params. 


#### Query Params options
    * Type
    * Title
    * Director
    * Cast
    * Country
    * Date_added
    * Release_year
    * Rating
    * Duration
    * Listed_in
    * Description
